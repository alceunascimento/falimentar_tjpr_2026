# falimentar_tjpr_2026

## Descrição
Apuração de dados sobre ações de falência, recuperação judicial e extrajudicial no Tribunal de Justiça do Estado do Paraná

## Estrutura
- `data/`    — dados brutos e processados
- `bin/`     — scripts de análise
- `outputs/` — resultados, tabelas, figuras
- `misc/`    — documentos auxiliares
